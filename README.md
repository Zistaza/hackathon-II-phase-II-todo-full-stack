# Todo Application - Authentication & Identity

## Overview
This is a full-stack todo application featuring secure authentication and identity management using Better Auth for frontend authentication and JWT-based backend verification.

## Features
- User registration and login
- JWT-based authentication
- Multi-tenant data isolation
- Secure API access

## Tech Stack
- Frontend: Next.js 16+, Better Auth
- Backend: Python FastAPI
- Database: Neon Serverless PostgreSQL
- ORM: SQLModel
